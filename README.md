## 📌 Project Overview

Employee attrition is a significant challenge for organizations as it impacts productivity, increases recruitment costs, and affects workforce stability.

This project applies Machine Learning techniques to analyze employee data and predict employee attrition. Multiple classification models were trained and evaluated to identify the most effective approach for workforce retention analytics.

---

## 🎯 Problem Statement

Organizations often struggle to identify employees who may leave the company in the future.

The goal of this project is to:

✅ Analyze employee attrition patterns

✅ Perform data preprocessing and feature engineering

✅ Train multiple machine learning models

✅ Compare model performance

✅ Generate actionable business insights
